## 📝 주요 파일 설명

### 1. 웹 페이지 파일들
- `app/(dashboard)/page.tsx` - 대시보드 홈 (어르신 리스트)
- `app/(dashboard)/elders/page.tsx` - 어르신 목록
- `app/(dashboard)/elders/[id]/page.tsx` - 어르신 상세 페이지
- `app/(dashboard)/elders/[id]/sessions/new/page.tsx` - 인터뷰 시작
- `app/(dashboard)/elders/[id]/sessions/[sessionId]/page.tsx` - 세션 상세

### 2. API 파일들
- `app/api/chat/route.ts` - GPT와의 대화 API
- `app/api/sessions/end/route.ts` - 세션 종료 및 요약 생성 API

### 3. 핵심 로직 파일들
- `lib/risk.ts` - 위험도 계산 엔진 (키워드 분석)
- `lib/openai.ts` - OpenAI 클라이언트 설정
- `lib/auth.ts` - 인증 관련 함수

### 4. 데이터베이스
- `supabase/migrations/001_initial_schema.sql` - 초기 스키마
- `supabase/migrations/002_improve_schema.sql` - 개선된 스키마
